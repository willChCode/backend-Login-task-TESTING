# Backend de Registro y Gestión de Tareas

Este es el backend de una aplicación para el registro y gestión de tareas. Proporciona funcionalidades de autenticación de usuarios, registro de tareas y gestión de sesiones utilizando diversas tecnologías y herramientas.

## Tecnologías utilizadas

- Testing con JEST: Framework de pruebas unitarias para asegurar la calidad del código.
- JWT (JSON Web Tokens): Método de autenticación y generación de tokens para proteger las rutas y mantener la sesión de usuario.
- BCRYPT: Librería para el hashing y verificación segura de contraseñas de usuarios.
- MongoDB: Base de datos NoSQL utilizada para almacenar la información de usuarios y tareas.
- Express.js: Framework de aplicaciones web utilizado para crear la API RESTful.
- Node.js: Entorno de ejecución para JavaScript utilizado en el servidor.

## Características

- Registro de usuarios: Permite a los usuarios crear una cuenta proporcionando su nombre de usuario y contraseña.
- Inicio de sesión: Los usuarios pueden iniciar sesión con sus credenciales para acceder a las funcionalidades protegidas.
- Gestión de tareas: Permite a los usuarios crear, leer, actualizar y eliminar tareas asociadas a su cuenta.
